# Bare Bones

A bare-bones example of the [Graph](/graph/Graph) component.

It may be used this way if, for instance, a force-directed layout computes the related geometry. Otherwise, these are uncommon:

- Both the `width` and `height` properties are provided.
- Connection terminals are _points_ rather than _ids_.

See [the next demo](/demos/some-flesh) for a more common usage.

## Source

```jsx
import React from 'react';
import { Graph } from '@regraph/graph';
import { Line } from '@regraph/connections';

export default () => (
  <Graph
    width={220}
    height={100}
    nodes={[{ id: 'ping' }, { id: 'pong' }]}
    boxes={{
      ping: { x: 50, y: 40, width: 20, height: 20 },
      pong: { x: 150, y: 40, width: 20, height: 20 },
    }}
    renderSvgNode={({ id, box }) => <rect key={id} {...box} fill="Chocolate" />}
    connections={[
      {
        id: 'ping->pong',
        src: { x: 60, y: 50 },
        dst: { x: 160, y: 50 },
      },
    ]}
    renderConnection={props => <Line key={props.id} {...props} />}
  />
);
```
